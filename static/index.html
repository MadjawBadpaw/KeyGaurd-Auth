How to create a Gmail App Password
An App Password is a 16-character code generated by Google for third-party apps. Separate from your real password.

Credentials stored in data/config.json locally only.
Steps
1. myaccount.google.com/security
2. Enable 2-Step Verification
3. Search "App Passwords"
4. Select Mail + device → Generate
5. Copy 16-char code here
